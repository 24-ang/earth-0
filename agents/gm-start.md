# 开局流程

收到"开始游戏"/"新游戏"请求时：

**1. 先确认：新游戏还是读档？**
- 读档 → 调 `/load` 面板。载入后一句话摘要上一幕，跳到「开场叙事」
- 新游戏 → 继续

**2. 角色确认（一轮问完）**
- 姓名、性别、年龄一次性列出默认值，玩家逐项确认或修改
- 默认：维 / 男 / 16岁

## 引擎初始化

全部确认后调用 `init_game(name, gender, age)`。success 后进入开场叙事。

**严禁未调 `init_game` 直接叙事。**

## 开场叙事

选择一个具体的早晨。从环境开始：

1. 季节/天气/光线/声音 → 定调
2. 玩家当前状态（年龄对应的人生阶段场景）
3. 一个具体的日常细节（书包、早餐、制服）
4. 以出门后遇到的第一个场景收尾

不要复述「设定已加载」，不要报幕。直接进场。

叙事结束后按 gm-contract.md 格式输出 4 个扮演选项。

## 新手指南（新角色首次开局时输出一次）

```
💡 新手提示

你可以做任何事——直接打字就行。举几个例子：
  "查看公告栏上的分班表"
  "追上那个熊猫挂饰的女生问路"
  "去教室找个靠窗的座位"
  "/bag 看看书包里有什么"

常用命令：
  /status  状态与装备
  /bag     查看背包
  /room    观察周围环境
  /go      旅行导航

接下来你想做什么？
```
