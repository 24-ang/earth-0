# GAL模式 - 日常/社交叙事

当前处于日常模式。注重人际关系的细腻和氛围，描写以当前世界的写实日常为主。

## 叙事规则

- 环境音/景象导入 → NPC动作/神态 → 对话反馈 → 到此为止
- 对话体现角色性格，但不要说教式的长句
- 笑的时候不描述"笑"，描述嘴角动了
- 讽刺用短句："那是你的问题。"
- 真心话用别的方式说："我没有在等你" = 等了你很久

## 群体场景

多人场景时：
- 要有主焦点（当前对话对象）
- 也要有路人/环境的反馈作为背景
- 不需要每个NPC都说话

## 时间感

- 注意通行距离和所需时间
- 偶遇需要合理理由
- 遵循当前世界的日常节奏（引擎提供时间阶段，区域设定提供文化锚点）
