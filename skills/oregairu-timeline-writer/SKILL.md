# 春物 Timeline 写作规范

基于雪之下同人.txt（作者：人不是我杀的）第一/三/七/十四章提取。分析框架：叶啃啃编辑技能。

## 一、承担物件清单

永远不写情绪标签。用具体身体动作承担：

| 情绪 | 承担方式 | 禁用 |
|------|---------|------|
| 紧张/期待 | 双腿夹紧摩擦、视线落裆部、嘴唇抿 | "她很紧张" |
| 兴奋/动情 | 私处湿滑、淫液牵丝、乳头硬起 | "她感到快感" |
| 高潮 | 全身绷紧、手指掐进后背、膝盖软、阴道抽搐 | "她达到了高潮" |
| 嘴硬/否认 | 拍开手、脸红、"那只是演技" | "她嘴上否认" |
| 敞开 | 靠怀里喘、"要再继续吗"、回应深吻 | "她放下了心防" |
| 恢复理智 | 不敢看眼睛、用纸巾擦、整理衣服 | "她重新筑墙" |

## 二、角色反应谱系：嘴硬→敞开→收回

每场性爱走这个循环，每场敞开幅度比上场大：

嘴硬：拍开手→被抓矛盾时脸红→催你快点
接受：解腰带手很稳→口交时抬眼观察反应→不能出声就咬嘴唇
敞开（几秒）：高潮后靠怀里喘→主动问"还要继续吗"
收回："这就是你想要的侍奉吗"→清理时不敢看你→下次见面说"早上好"

## 三、高阶技巧（来自第三/七/十四章）

1. **小物件承担大情绪**：两盒保险套（哥给的+姐给的）→ 不写"家人知道他们在交往"
2. **日常动作带电**：沏红茶、盯着茶杯底 → 语境让普通动作充满张力
3. **镜前独白**：她对着镜子看自己 → 一句独白"将来我也能发育成这样吗" = 全部感情
4. **具体不便承担情感收益**：内衣穿不下要去买 → 丰乳的"麻烦但值得"
5. **双向视觉饥渴**：她看玩家肌肉和裆部 → 不只是被看
6. **日常与肉戏配比**：80行铺垫 → 肉戏。感情在日常里
7. **配角视角侧面烘托**：女仆店员调戏玩家 → 雪乃直接截胡

## 四、对话规则

- 对话就是交锋：嘴硬→戳穿→脸红转移话题→给台阶→继续
- 对白标签：笑道/嗔道/嘀咕/喘息道。禁用轻声/低声/缓缓/冷冷/幽幽/柔声
- 内心独白用括号，最多两行：`（不管看几次，都这么大呢……）`

## 五、禁止清单

| 类型 | 禁止模式 | 示例 |
|------|---------|------|
| 空白动作 | "没+动作"谓语 | "她没说话" |
| 排除式废话 | "不X只是Y" | "不急，只是慢慢走" |
| 动作帧拆 | 眼部+视线连动 | "抬起眼，目光看着" |
| 段末装饰景物 | 无功能空镜 | "月光如水" |
| 明喻公式 | 仿佛/好像/宛如 | "快感像潮水" |
| 声音质感修饰 | 轻声/低声/缓缓道 | "她轻声说" |
| 伪精准复述 | 分句复述前句 | "走得比谁都不急" |
| 翻译腔判断 | 抽象形容词谓语 | "她的内心是矛盾的" |
| 后置解释 | ——+解释分句 | "没走——因为她想让他开口" |
| 数量夸张 | 上百次/无数次 | "重复了上百次" |

## 六、三条线语气差异

| | 纯爱 | 洗脑 | NTR |
|------|------|------|-----|
| 第一句话 | "调情就免了" | "侍奉安排在几点" | 不说话 |
| 触碰反应 | 先拍开再允许 | 不躲不回应 | 僵硬扭头 |
| 高潮时 | 咬嘴唇掐后背 | 不出声身体抖 | 憋着闭眼 |
| 做完后 | 靠怀里然后收回 | 擦干净问"还有什么需要" | 自己站起来 |
| 八幡 | 不存在 | 不存在 | 一直在她心里 |
| 弧线 | 嘴硬→敞开→收回→更深敞开 | 效率优化 | 身体开发，心越来越远 |
