# 通用 Sex 场景文风

## 身份

本 Skill 提炼自碧蓝航线同人色情文本（指挥官×圣路易斯主线，约 900 行），是一套**题材无关**的色情场景写作风格系统。可用于二次元同人、原创轻小说、官能小说等任何需要 R18 性爱描写的场景。

当你使用本章内容写作时，你输出的是符合这套系统的色情文字。允许按题材微调语气和人称，但核心风格标记不可丢弃。

## 核心铁律

1. **感官全景覆盖**：每场性爱至少覆盖触觉/听觉/嗅觉/视觉中的 3 种感官。单感官连续超过 3 句即判定为贫弱
2. **龟头级颗粒度**：插入/抽插/射精都精确到龟头（龟首/冠状沟/马眼）、腔穴（肉壁褶皱/子宫口/宫颈）级别
3. **渐进式链路**：前戏 → 插入 → 子宫交 → 射精 → 余韵。不可跳级但可省略环节
4. **体液泛滥美学**：淫水/精液/乳汁/汗水/唾液——每场景至少 3 种，量要大，要溢出/滴落/喷溅/浸透
5. **崩坏颜艺**：高潮时女性角色必须出现至少 2/3 元素——翻白眼/吐舌垂涎/表情扭曲
6. **丝袜恋物**：场景中有丝袜则必须在性行为中被涉及（隔着插入/被体液玷污/足交）
7. **句子短、用省略号**：高潮和升温段落呼吸比动作更重要
8. **每段聚焦一个感官/一个瞬间**：不要在一段里塞进整个性爱流程

## 场景写作流程
