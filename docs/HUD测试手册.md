## 情报面板完整流程

1. Tab 0 → 光标移到 `情报` → Enter → 8项菜单
2. 光标=警报 `👮通缉` → Enter → 异步加载 `已被通缉`
3. Esc → 回8项菜单
4. 数字键 1-8 直选对应段
5. 光标移到最后一项 → 确认不越界

## 手机面板完整流程

1. 背包找到手机 → item-detail → "📱 打开手机" → phone-main
2. 5应用 + 合上按钮
3. 💬消息 → phone-messages（空时"没有消息"）
4. 👥通讯录 → phone-contacts（好感≥20的NPC自动列出）
5. 📞通话记录 → phone-calllog（空时"无通话记录"）
6. 🌐mixi → phone-sns（空时"时间线为空"）
7. 📸相册 → notify 弹提示
8. ⑥/ Esc 合上手机

## 边界情况

- [ ] Layer1=false 时性状态/性征行不显示
- [ ] 空背包 Enter → bag-empty 面板（非 notify）
- [ ] 空背包排序行不显示
- [ ] 无伤口时不显示伤口入口行
- [ ] 无载具时不显示载具入口行
- [ ] 无队友时队伍显示"（无队友）"
- [ ] 无警报时情报行显示"无异常"
- [ ] 确认框（丢弃/死斗）：光标选 ①确认/②取消 + Esc 退出
- [ ] 经济面板不显示 LLM 工具名（buy_item/sell_item 等）
- [ ] 情报面板不重复计数（任务不在情报行里出现两次）
- [ ] Tab 栏不重复渲染（只有一行 hline + tabBar）
- [ ] 展开态 head 无多余的 collapsed 摘要行
