- **爱莉好感10但 stage="陌生"**：关系阶段由叙事推进，引擎不强制覆盖
- **35岁 isVirgin: true**：引擎不替 LLM 做性经历判断（PHILOSOPHY §1.2）
- **手机联系人**：init_profile 末尾自动从已建立的关系同步联系人（无好感门槛）
- **职业身份**：`set_flags(is_teacher=true)` 后引擎自动授予"教师"称号
- **NPC 移动是瞬移**：日程切换时 NPC 瞬间换位（非实时寻路），但门关了/玩家堵门/家具堵出口 → NPC 被拦住
