## 二、详细剧情（全三话 + 番外）

### 第一话「只有我不能做爱的房子」

**场景1：课题小组**
- 次男（优斗）在班上跟朝陽（黑长直眼镜优等生）和芽衣（金发辣妹）分到同一个课题小组。
- 次男暗恋朝陽，觉得这是天赐良机——邀请两人来家里一起做课题报告，两人答应了。
- 次男内心激动：这是第一次有女生来他家。

**场景2：城户家客厅**
- 朝陽和芽衣来到城户家。次男紧张地招待。
- 大哥（城户树）正好在家——戴着眼镜的学生会长，高大帅气。三弟（城户状真）也在——金毛チャラ男，国三但体格健壮。
- 朝陽看到大哥后明显变得矜持紧张——原来她一直暗恋会长。芽衣也被大哥吸引。
- 课题进行中，大哥自然地加入谈话——谈吐风趣、知识面广。朝陽的眼神逐渐从他身上移不开。
- 三弟则用没大没小的态度跟芽衣搭话——芽衣被他的直球逗笑了。

**场景3：当晚**
- 课题结束后天色已晚，大哥提议两人留宿——"这么晚了，女孩子回去不安全。"
- 朝陽和芽衣同意了。次男欣喜但不敢表现出来。
- 夜深后——大哥敲了朝陽的房门。三弟去了芽衣那边。
- 次男在自己房间打游戏——游戏通关了，他拿了第一。但他不知道两侧的房间里发生了什么。
- 分镜：左边→大哥和朝陽接吻。右边→三弟和芽衣在嬉闹中滚到床上。中间→次男盯着屏幕，戴着耳机，什么都没听到。

**场景4：第二天**
- 早餐桌上气氛微妙。朝陽不敢看次男的眼睛。
- 芽衣倒是大大方方——"你家的床好舒服！"
- 之后几天，次男发现朝陽和芽衣频繁来家里——不是找他，是找他兄弟。他在学校看到她们正常聊天，"就像什么都没发生过一样"——这让次男感到极度不适。
- 朝陽开始摘掉眼镜改戴隐形。芽衣的妆容比以前精致了。
- 终幕：次男在自己的房间里，隔墙听到朝陽在大哥房间里。他戴上耳机开最大音量——但隐约的声音还是穿透了音乐。

### 番外「派·芽衣Before朝陽After」

**Part A: 芽衣Before（前日谭）**

1. **中学一年级的芽衣** — "只跟喜欢的人做"是她的信条。她在走廊鼓起勇气向部長（学生会的前辈）告白。
2. **被拒** — 部長说："我有喜欢的人了。"芽衣尽力忍住眼泪，转身离开。
3. **趁虚而入** — 渋谷先輩目睹了整个过程。他追上去——"你还好吗？""他不值得你这样。"温柔但顺手把她拉进了空教室。
4. **失去处女** — 渋谷先輩说："做爱嘛，就是这样的。""比起哭，不如体验一下？"→芽衣没有反抗——更多的是麻木。结束后的渋谷先輩对着镜子露出得意的笑容："やったぜ。"
5. **在走廊角落** — 一个不起眼的男生（未来的"新主人公"）撞见了她和渋谷先輩从空教室出来。芽衣眼角红了。那个男生攥紧拳头："ふざけんな。絶対彼女作ってやる。"——这是前篇和后篇之间无声的连接线。
6. **转折** — 从那之后，芽衣不再说"只跟喜欢的人做"。她变成了别人口中"轻い女"。但讽刺的是——在这整个系列里，她真正想一起睡的人（会长），她只在交换那一次得到过。

**Part B: 朝陽After（后日谭）**

1. **新学校，新男友** — 朝陽跟大哥分手后，转到另一所高中。交了一个新男友——他就是曾在走廊目睹芽衣被欺负的那个男生。他现在变帅了，真的交了女朋友——而且女朋友竟然是朝陽。
2. **坦白** — 朝陽对现男友坦诚："我……迷上做爱了。""可能你会觉得我脏。"现男友的回答："谁还没点过去啊。我以前也喜欢过一个女孩子，她跟我不喜欢的前辈在一起了。但我已经忘了。"
3. **罪悪感消散** — 朝陽提到次男转校的事，语气轻描淡写："有个人以前好像喜欢我……后来转校了。""他对我挺好，但我当时已经有男朋友了……"话没说完。现男友："那是他自己的问题，不关你的事。"→朝陽安心地靠进他怀里。次男的悲剧被永远埋葬。
4. **朝陽当选** — 最终朝陽升为学生会长。"现在我更喜欢我自己。"
